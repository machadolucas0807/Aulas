Imagem Dinamica

O elemento <picture> permite que você forneça diferentes imagens para diferentes tamanhos de tela. Isso é útil para otimizar o carregamento de imagens em dispositivos móveis e desktops.

O elemento <source> dentro do <picture> define as imagens que serão carregadas com base nas condições especificadas no atributo media.
