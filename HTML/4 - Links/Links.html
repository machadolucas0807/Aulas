2. Controlando onde o site abrirá

  • target="_blank": vai abrir o link em uma nova janela em branco.
  • target="_self": vai abrir o link na mesma janela (padrão).
  • target="_parent": similar ao uso do _top em uma referência à janela mãe.
  • target="_top": vai desfazer todos os frames e abrir o destino no navegador completo.
  • nome-do-frame: caso esteja usando frames, indica o nome da janela a abrir.

Exemplo:

  • Meu Instagram (target="_blank").
  • Meu Instagram (target="_self").
  • Meu Instagram (target="_parent").
  • Meu Instagram (target="_top").
  • Meu Instagram (target="janela").

3. Atributo rel

O atributo rel é usado para especificar a relação entre o documento atual e o link. Alguns valores comuns incluem:

  • next: indica que o link é para a próxima parte do documento atual;
  • prev: indica que o link é para a parte anterior do documento atual;
  • nofollow: indica que o link não deve ser seguido pelos motores de busca;
  • author: indica que é um link para o site do autor do artigo atual;
  • external: indica que é um link para outro site que não faz parte do site atual.

Exemplo:

  • Meu Instagram (rel="external").
  • Meu Instagram (rel="nofollow").
  • Meu Instagram (rel="author").
  • Meu Instagram (rel="prev").
  • Meu Instagram (rel="next").
