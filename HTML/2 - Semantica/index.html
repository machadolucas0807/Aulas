Vamos falar sobre semântica

Negrito

A tag strong significa que o termo delimitado possui força dentro da frase. Logo, ele aparecerá em negrito.

Itálico

A tag em significa que o termo delimitado possui ênfase dentro da frase. Logo, ele aparecerá em itálico.

Destacar

A tag mark é usada para destacar ou demarcar texto importante, geralmente com um fundo amarelo.

Texto Pequeno

A tag small é usada para indicar que o texto é menor que o texto padrão. Geralmente, ela é usada para notas de rodapé ou informações adicionais.

Texto Removido

A tag del é usada para indicar que o texto foi excluído ou removido. Geralmente, ela aparece com uma linha atravessando o texto.

Texto Inserido

A tag ins é usada para indicar que o texto foi inserido ou adicionado. Geralmente, ela aparece com um sublinhado.

Sobrescrito e Subscrito

A tag sup é usada para sobrescrever o texto, enquanto a tag sub é usada para subscrever o texto.

Exemplo de sobrescrito: E = mc2

Exemplo de subscrito: H2O

Trechos de Código

A tag code é usada para indicar que o texto é um trecho de código. Geralmente, ela aparece com uma fonte monoespaçada.

Exemplo de código: console.log("Hello, World!");

Tag Pre

A tag 

pre
é usada para indicar que o texto deve ser exibido em um formato pré-formatado. Geralmente, ela preserva espaços em branco e quebras de linha.

Exemplo de texto pré-formatado:

        function helloWorld() {
            console.log("Hello, World!");
        }

Citações

  • A tag

    blockquote
    é usada para indicar que o texto é uma citação de outra fonte. Geralmente, ela aparece com uma margem maior.

    Exemplo de citação:

    "A vida é o que acontece enquanto você está ocupado fazendo outros planos." - John Lennon

  • A tag cite é usada para indicar que o texto é uma citação de outra fonte. Geralmente, ela aparece com uma fonte itálica.

    Exemplo de citação:

    A vida é o que acontece enquanto você está ocupado fazendo outros planos. - John Lennon

  • A tag q é usada para indicar que o texto é uma citação de outra fonte. Geralmente, ela aparece com aspas ao redor do texto.

    Exemplo de citação:

    A vida é o que acontece enquanto você está ocupado fazendo outros planos. - John Lennon

  • A tag abbr é usada para indicar que o texto é uma abreviação. Geralmente, ela aparece com uma fonte itálica.

    Exemplo de abreviação:

    HTML
